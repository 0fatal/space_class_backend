# 班级空间 - 后端

## 目录结构

```sh
├── api    # 定义处理函数
├── db       # 数据库连接配置
├── main.js      # 启动入口文件
├── node_modules
├── package-lock.json
├── package.json
├── router        # 请求路由定义
├── middleware    # 中间件
├── dto          # 返回包装器
├── test          # 单元测试
├── utils        # 工具函数
└── service      # 基本功能函数
```

## 如何运行项目

1. `npm install`
2. `npm run start`
